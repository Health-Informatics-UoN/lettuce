# Lettuce API

| Component | Description |
|:----------|------------:|

|[`components`](/api_reference/components)| Natural language processing pipeline components |

|[`options`](/api_reference/options)| Sets options for the CLI and HTTP API |
|[`routers`](/api_reference/routers)| API endpoints for HTPP requests |
|[`utils`](/api_reference/utils)| Miscellaneous utilites for Lettuce |
|[`app`](/api_reference/app)| The FastAPI app to provide endpoints for requests |
|[`assistant`](/api_reference/assistant)| Executes a simple assistant |
