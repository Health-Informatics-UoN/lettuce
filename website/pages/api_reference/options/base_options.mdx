# `options.base_options`

[source](https://github.com/Health-Informatics-UoN/lettuce/tree/main/lettuce/options/base_options.py)

This module contains the `BaseOptions` class defines command-line options used during all types of experiments in `lettuce`. 
It handles initialisation, parsing, and display of various configuration options, particularly focusing on LLM model settings, embedding configurations, and OMOP vocabulary filtering parameters.

## `BaseOptions`
```python
class BaseOptions()
```

A utility class that manages command-line arguments for configuring the `lettuce` pipeline. 

### Methods

#### `initialize`
```python
def initialize()
```

Initialises the `BaseOptions` class. Sets up all available command-line arguments with their default values and help text.

#### `parse`
```python
def parse()
```

Parses the arguments passed to the script and prints the configuration.

##### Returns
- `opt: argparse.Namespace`

	The parsed arguments
