## OMOP

This module details Lettuce's interactions with the OMOP-CDM database

| component | description |
|:----------|------------:|
|[`db_manager`](/api_reference/omop/db_manager)| Provides session management functionality for connecting to the OMOP-CDM via SQLalchemy |
|[`omop_match`](/api_reference/omop/omop_match)| Class and function for fetching concepts from the OMOP-CDM database |
|[`omop_models`](/api_reference/omop/omop_models)| ORM models for tables in the OMOP-CDM database |
|[`omop_queries`](/api_reference/omop/omop_queries)| Provides a set of functions which construct queries to the OMOP-CDM database |
|[`preprocess`](/api_reference/omop/preprocess)| Text preprocessing for use in text search |
